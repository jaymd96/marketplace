# new thread

Create a new idea thread when a line of reasoning emerges.

## Trigger

- A new idea worth tracking across sessions
- "Let's track this idea"
- "This is a separate line of thinking"

## What to do

Given a thread name:

1. Create `researcher/threads/<name>/` directory
2. Create `raw-notes.md` with header
3. Create `connections.md` with header (links to papers, other threads)
4. Create `questions.md` with header
5. Create `resolved.md` with header
6. Create `status.md` with: `active`, creation date, key question
7. Add to Active Threads table in PROJECT_STATE.md
8. Add to THREAD_MAP.md

Don't create threads for passing thoughts. A thread is a line of reasoning
worth tracking across sessions.
