# Spec Builder Evolution Changelog

Record of all changes to the spec-builder system itself.

Changes here affect ALL future projects and sessions. Each entry references
the evolution proposal (EP) that drove it.

---

## 2026-03-07 — System Created

**What:** Initial spec-builder system: 11 workflows, 5 templates, entrypoint,
git integration, self-review, self-evolution protocol.
**Why:** Need a structured system for building product specs through extended
multi-session human conversation. Must survive context loss, handle noisy input,
detect inconsistencies, and produce Symphony-style specifications.
